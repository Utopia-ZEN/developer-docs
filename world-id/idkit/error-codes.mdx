## Handling errors

Widgets expose an `onError` callback. Hooks expose `isError` and `errorCode` on the result object.

```tsx
<IDKitRequestWidget
  // ...
  onError={(errorCode) => {
    console.error("IDKit error", errorCode);
  }}
/>
```

```tsx
const flow = useIDKitRequest({ /* ... */ });

if (flow.isError) {
  console.error(flow.errorCode);
}
```
